# Open Source Alternative to Zyte

Zyte (formerly Scrapy Cloud) offers managed Python/Scrapy hosting. **Crawlee Cloud** lets you run TypeScript scrapers on your own cloud.

## Quick Comparison

| Feature | Zyte | Crawlee Cloud |
|---------|------|---------------|
| **Language** | Python/Scrapy | TypeScript/Crawlee |
| **Hosting** | Zyte Cloud | Self-Hosted |
| **Anti-Bot** | Smart Proxy Manager | Bring Your Own Proxy |
| **Source Code** | Closed | Open Source |
| **Pricing** | Pay-per-request | Your infrastructure costs |

## Why Choose Crawlee Cloud?

- **TypeScript-first** — Modern async/await patterns
- **Self-hosted** — Full control over your infrastructure
- **Any proxy** — Use any proxy provider you want
- **Open source** — MIT licensed, fully customizable
