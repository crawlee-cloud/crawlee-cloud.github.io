# Self-Hosted Alternative to ScrapingBee

ScrapingBee provides a headless browser API. **Crawlee Cloud** lets you run your own headless browsers at scale — no per-request fees.

## Quick Comparison

| Feature | ScrapingBee | Crawlee Cloud |
|---------|-------------|---------------|
| **Model** | API calls | Your own scrapers |
| **Flexibility** | Limited to API | Full code control |
| **Data** | Passes through their servers | Never leaves your infra |
| **Cost at Scale** | Grows with usage | Fixed VPS cost |

## Why Choose Crawlee Cloud?

- **No per-request fees** — Fixed infrastructure cost
- **Full control** — Write any scraping logic you need
- **Data privacy** — Scraped data never leaves your servers
- **Any browser** — Playwright, Puppeteer, or headless options
